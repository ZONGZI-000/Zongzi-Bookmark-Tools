WebDAV 设置

WebDAV URL、用户名和密码会加密保存在本机浏览器存储中。

Git 同步设置

令牌仅保存在本机加密存储中；上传会提交 JSON 快照，仓库提交记录即版本历史。

本地文件导入/导出

无需网络。导出会生成当前书签 JSON 备份；导入时会按下方下载模式恢复。
